This project has three stage


First: apply 10 fold cross validation on UrbanSound8K dataset for classification using the model described in this paper (base paper)
"Environmental Sound Classification for Normal Hearing and Hearing-Impaired Individuals Using a Hybrid CNN-BiLSTM-Attention Model"
Authors :Md.Fariaz Islam,Sarkar Minhaz Ahmed, Dr. Nursadul Mamun,Robust Speech Processing Laboratory (RSPL),Department of Electronics and Telecommunication Engineering, Chittagong University of Engineering and Technology, Chittagong

Second: use Vision Transformer(ViT) on the same dataset for classification

Third build a Generative Adversarial Network (GAN) model for classification on the same dataset.
